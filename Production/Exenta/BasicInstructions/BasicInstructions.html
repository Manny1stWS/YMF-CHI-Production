EXENTA BASIC PRODUCTION INSTRUCTIONS

Prepared by Manuel Herrera

Updated: 2/17/24

STYLE MASTER

You can access the Style Master by going to: Reference>Style>Style Master:

You can create a new Style by clicking the “New” icon on the top left side of the window:

After clicking “New” icon , some fields will populate automatically:

  1. Company
  2. Division
  3. Season
  4. Status
  5. UOM
  6. Purchase UOM
  7. Size

Mandatory Fields to Enter:

Data Tab:

  1. Style (This is where the Style# is entered)
  2. Long Description (100 characters limit)
  3. Style Name (30 characters limit)
  4. Brand (right click to select)
  5. Department (right click to select)
  6. Category (right click to select)
  7. Sub-Category (right click to select)
  8. Const Type (right click to select)

If you are not sure what is the Department, Category, Sub-Category and/or Const Type of your Style, please speak to the Product Develpment person associated to your Style and if you don’t that either, speak to the production manager for direction.

Additional 1 Tab:

  1. PK Type (right click to select)
  2. IN Type (right click to select)
  3. CS Pack (right click to select)
  4. HS Number (right click to select)
  5. Vendor (right click to select)
  6. Country of Origin (right click to select)

Style Detail:

  1. Click the “New” icon
    • Style Color window will popup.
  2. Color (right click to select)
  3. Size Code (right click to select)
    • Be sure to select the code with same prefix as Category Code.
  4. PPK
    • This is used only for Prepack Styles. Will explain in more detail below.
  5. Size Allow (To enter UPC Code)
    • Style Dim/Size window will popup.
  6. Click Save to finish.

In Style Detail it is very important that you only have one line per Style with correct Size Code and/or PPK. Do not add multiple lines without permission from Greg Tsirulnik. If Multiple lines already exists do not delete lines without asking Greg Tsirulnik. And Do Not use the “Label” field.

Prepack Style Setup

Before entering a Prepack Style in the Style Master you must first enter the individual styles in the Style Master and then setup a Prepack by going to Reference>Style>Prepack

  1. Click the “New” Icon
  2. PPK: Enter the Prepack Style number. This number will be the same as the new Style# you will add in the Style Master.
  3. Size Range: Right click to select.
  4. Prepack Desc: Enter the description of the Style.
  5. Asst Type: Chose Style to enter prepack by Style number.
  6. Add: click to Add individual Style numbers to the prepack.
  7. Division: Right click to select CHI.
  8. Style: Paste Style number part of the prepack.
  9. Color: Right click to select.
  10. Size Code: Right click to select.
  11. Size: Right click to select.
  12. PPK Ratio: Enter number of pieces of this style in the prepack.
  13. Repeat steps 6 to 12 up to you finish entering all the styles in the prepack.
  14. Click Save to finish.

Production Order Maintenance

You can access Production Order Maintenance by going to: Application>Production Order Management>Production Order Maintenance

You can create a new Purchase Order by clicking the “New” icon on the top left side of the window:

After clicking “New” , some fields will populate automatically:

  1. Company
  2. Prod Category
  3. Division
  4. Season
  5. Pay to
  6. Prod Owner
  7. Prod Order Date
  8. Printed
  9. Release Status

Mandatory Fields to Enter:

  1. Prod Type (Right click to select) – (always select YMF-GEN)
  2. Vendor (You can Right click to select) to find a vendor you can either scroll thru the vendors or you can click on top of one of the Vendor Name and then quickly type the Vendor Name you are looking for or you can click on Search Tab, type a portion of the vendor’s name in between two *stars* then click find and then the “Zoom” window will display only the vendors with the word you typed for you to select from.
  3. Warehouse (Right click to select). See below Warehouse code list and what each code means.
  4. Ship Via (Right click to select). You can select TBD1.
  5. Shipping Port (Right click to select). Make sure to select the correct port the order will ship from.
  6. Port of Entry (Right click to select). Make sure to select the correct port the order will ship to.
  7. ReferenceNo. Enter the customer PO# related to your PO if applicable.
  8. Customer (Right click to select).
  9. Origin ETD (Right click to select). Select the date you want the factory to ship your PO.
  10. Expected Receipt (Right click to select).
  11. Click Save to get a PO# in the “Prod Order No” field.

Entering Styles in PO

Go to “Prod Order Detail” section and then click “New” then the “Prod Order Detail” Window will popup.

In the Prod Order Detail Window, you must enter the following fields:

  1. Style (Copy and Paste the Style number you want to enter. If you right click, it will take a few seconds for all the style numbers in the system to show up in a different window so I suggest to never right click if you can avoid it)
  2. Color (This should populate automatically but if it doesn’t, you can right click to select “NC” since as of now this is the only option for color)
  3. Size Code (Right click to select)
  4. PPK (Right click to select only if applicable)
  5. FOBCost (Enter the price per piece)
  6. Prod Line Qty (Enter the number of Pieces you are ordering)
  7. You can Click Next to enter another Style or OK if you finish entering all Styles to go back to your PO.
